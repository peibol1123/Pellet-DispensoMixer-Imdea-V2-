Firmware Compilation and Upload

This document describes the procedure used to compile and upload the Pellet DispensoMixer V2 firmware.

The firmware was developed using the Arduino IDE and uploaded to the BIGTREETECH Octopus Pro V1.1 control board.

Required Software

Before compiling the firmware, install the following software:

Arduino IDE

Download and install the Arduino IDE:

https://www.arduino.cc/en/software
